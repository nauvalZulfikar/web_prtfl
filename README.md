# web_ptfl — Nauval Zulfikar Portfolio

Production-ready personal portfolio website untuk Nauval Zulfikar (Senior Data Scientist).

**Aesthetic:** Refined editorial minimalism — high-end consultancy / serious research publication, bukan startup landing page.

## Stack

- Next.js 14 (App Router)
- Tailwind CSS
- TypeScript

## Deploy

Target: **Vercel**.

## Run

```bash
pnpm install   # atau npm install
pnpm dev       # http://localhost:3000
pnpm build
pnpm start
```

## Layout

```
web_ptfl/
├── app/             Next.js App Router pages
├── components/      Shared components
├── lib/             Utilities
├── public/          Static assets (CV PDF, dll)
└── AGENTS.md        Detailed brief — aesthetic, content blocks, sections
```
